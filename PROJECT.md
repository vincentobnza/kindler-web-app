Kindle Reader Web Application
Project Overview

The Kindle Reader Web Application is a browser-based platform that allows users to discover, search, view, and save books using the Open Library API.

The application is built using React and does not require a backend server. All book data is fetched directly from the Open Library API, while saved books are stored locally in the browser using Local Storage.

Objectives
Browse available books
Search books by title
View book details
Display book cover images
Save favorite books locally
Provide a simple Kindle-like reading experience

API
Open Library API

Base URL

https://openlibrary.org

Search Endpoint

https://openlibrary.org/search.json?q=

Cover Images

https://covers.openlibrary.org/b/id/{cover_i}-L.jpg
